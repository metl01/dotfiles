# dotfiles
Download these fonts:
```sh
sudo pacman -S ttf-firacode-nerd ttf-adwaitamono-nerd
```
Dependencies:

Hyprland,
Hyprlock,
Hypridle,
Hyprshot,
Hyprpicker,
Hyprpaper,
Waybar,
Rofi,
Imagemagick,
and Kitty
