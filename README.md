# dotfiles
Download these fonts:
```sh
sudo pacman -S ttf-firacode-nerd
yay -S ttf-adwaita-mono-nerd
```

Hyprland,
Hyprlock,
Hypridle,
Hyprshot,
Hyprpicker,
Hyprpaper,
Waybar,
Rofi,
Imagemagick,
Power Profiles Daemon,
